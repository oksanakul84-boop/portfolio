Про мене

Вітаю! Я щаслива мама, любляча дружина та жінка, яка поєднує сімейні цінності з прагненням до розвитку. Маю аналітичне мислення, відповідальність та увагу до деталей. Маю дозвід роботи з ERP-системами, управлінням документами та оптимізацією робочих потоків. Легко та швидко освоюю нові програми, люблю аналізувати інформацію, знаходити рішення та наводити порядок у складних системах. У вільний час подорожую, вивчаю польську мову, читаю, займаюсь йогою та фітнесом. Мене надихає розвиток, нові знання та рух до гармонійного життя. Зараз активно навчаюся в Академії IT LIFE Україна за напрямком QA-тестування, відкриваючи для себе нові професійні можливості.

Професійні навички
System & Process Analys
Аналіз та формалізація бізнес-вимог, переклад у технічні вимоги. Опис та оптимізація бізнес-процесів (AS-IS / TO-BE) Підготовка функціональних та нефункціональних вимог. Участь у впровадженні та супроводі інформаційних систем.
Data & Process Analys
Аналіз даних для підтримки управлінських рішень. Виявлення проблем у роботі систем та процесів Пошук рішень для підвищення ефективності систем. Контроль коректності обробки та збереження даних.
System Implemention & Support
Участь у впровадженні ERP та інших корпоративних систем. Взаємодія з розробниками, тестувальниками та користувачами. Аналіз інцидентів та підтримка користувачів. Тестування змін та оновлень систем.
Documentation & Modeling
Підготовка технічної та користувацької документації. Опис вимог, інструкцій, регламентів. Створення схем і моделей процесів Формування специфікацій для розробників та тестувальників.
QA & Testing
Розуміння життєвого циклу розробки ПЗ (SDLC, STLC). Ручне тестування веб- та мобільних застосунків. Функціональне, регресійне, smoke та exploratory testing. Аналіз вимог та пошук невідповідностей. Виявлення, опис та відстеження багів.
Web Basics
Базові знання HTML, CSS. Розуміння клієнт-серверної архітектури. Робота з браузерними DevTools.
API & Tools
Базове розуміння API. Тестування API (Postman, Swagger — базовий рівень).
